Assume you have jQuery or Prototype, write performance-wise script to:

hide all even rows
// Hide all rows with the 'even' class $('.even').hide();
do same assuming there are no class names
// Use 'odd' because indexes are 0-based $('li:odd').hide();
Build two arrays containing odd and even rows respectively.
// Even elements var even = $('li:odd'); // Odd elements var odd = $('li:even');
Rearrange nodes in reverse order
// your js code var listItems = $('li'); for( var i = 0; i < listItems.length; i++ ) { $('ul').prepend(listItems[i]); } // Alternatively var listItems = $('li').get().reverse(); $('ul').html(listItems);
